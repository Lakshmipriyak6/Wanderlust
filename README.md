# Wanderlust 

A travel accommodation platform with innovative booking experiences.

## Project Overview

Wanderlust is a full-stack web application inspired by Airbnb that enables users to discover, create, manage, and review accommodation listings. The application provides secure user authentication, image uploads, location-based services, and a user-friendly interface for managing travel accommodations.

## Unique Features

### Ghost Listing

Ghost Listing is an innovative feature that introduces mystery-based travel booking. Selected properties appear as hidden or partially revealed listings, allowing users to unlock exclusive deals through a surprise booking experience.

#### How It Works
- Property images are blurred until booking is completed.
- Exact location details remain hidden from users.
- A countdown timer displays the remaining time before the offer expires.
- Listings are available at significantly discounted prices compared to regular bookings.
- After a successful booking, all property details, images, and location information are instantly revealed.

#### Why It's Unique
- Creates excitement and curiosity during the booking process.
- Encourages users to discover new destinations.
- Provides access to exclusive discounted accommodations.
- Adds a gamified experience to traditional travel booking platforms.

---

### Carbon Passport

Carbon Passport is a sustainability-focused feature that helps users track and understand the environmental impact of their travel decisions.

#### How It Works
- Users provide their travel distance and mode of transportation (Flight, Train, Car, Bus, etc.).
- The system calculates the estimated carbon emissions generated by the trip.
- A cumulative carbon footprint score is maintained for each user.
- Users can plant virtual trees to offset their emissions and improve their sustainability score.
- A live CO₂ tracker is displayed within the platform to promote environmentally conscious travel.

#### Features
- Real-time carbon emission calculation.
- Personal carbon footprint tracking.
- Virtual tree planting and carbon offset system.
- Sustainability-focused user engagement.
- Environmental awareness through travel analytics.

#### Why It's Unique
- Encourages responsible and eco-friendly travel habits.
- Provides users with insights into their environmental impact.
- Combines travel planning with sustainability awareness.
- Introduces a meaningful gamification element beyond traditional booking features.

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/wanderlust.git
cd wanderlust
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory and add the following:

```env
ATLASDB_URL=your_mongodb_connection_string

CLOUD_NAME=your_cloudinary_cloud_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret

MAP_TOKEN=your_mapbox_access_token

SECRET=your_session_secret
```

### Run the Application

```bash
node app.js
```

or

```bash
nodemon app.js
```

### Open in Browser

```text
http://localhost:8080
```
